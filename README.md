This project was developed with the following technologies:

- [Vite](https://vitejs.dev/)

## How To Use

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
